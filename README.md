# 🍦 Ice Cream Website

A simple and responsive **Ice Cream Website** featuring a ***modern menu bar*** with smooth navigation and mobile-friendly design.

---

## 📌 Features

* 🍨 Responsive navigation bar (desktop & mobile)
* 📱 Mobile menu (vertical toggle menu)
* 🎯 Smooth scrolling to sections
* ⬆️ “Go to Top” button
* 🎨 Clean and simple UI design
* ⚡ Lightweight and fast

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

---

## 📂 Project Structure

```
📁 Ice-Cream-Website
 ┣ 📄 index.html
 ┣ 📄 style.css
 ┣ 📄 script.js
 ┗ 📁 assets (images/icons)
```

---

## ⚙️ How to Run the Project

1. Clone the repository:

   ```
   git clone https://github.com/its-swaroopa2025/MENU-BAR-created.git
   ```

2. Open the folder

3. Run `index.html` in your browser
